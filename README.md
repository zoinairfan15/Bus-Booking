# 🚌 Bus Seat Booking System

A console-based + web-based Bus Seat Booking System built using Python and Flask.
Users can view the bus seat layout, book seats, and cancel bookings in real time.

---

## 🛠️ Tech Stack

- **Backend**  : Python, Flask
- **Frontend** : HTML, CSS, JavaScript
- **Deployment**: Render (backend) / GitHub Pages

---

## 📁 Project Structure

BusBookingWeb/
│
├── app.py                  ← Flask backend (routes + logic)
├── templates/
│   └── index.html          ← Frontend HTML + JavaScript
├── static/
│   └── style.css           ← CSS Styling
└── README.md               ← Project documentation

---

## ✨ Features

- 🚌 36 seat bus layout (9 rows × 4 seats)
- ✅ Book a seat by clicking on it
- ❌ Cancel a booked seat easily
- 👦 Child ticket      : Rs. 40
- 🧑 Adult ticket      : Rs. 80
- 👴 Senior Citizen    : Rs. 68 (15% discount on Rs. 80)
- 🟢 Available seats shown in GREEN with seat number
- 🔴 Booked seats shown in RED with ✕

---

## 🚀 How to Run Locally

### 1. Clone the repository
git clone https://github.com/your-username/BusBookingWeb.git

### 2. Go into the project folder
cd BusBookingWeb

### 3. Install Flask
pip install flask

### 4. Run the app
python app.py

### 5. Open in browser
http://127.0.0.1:5000
